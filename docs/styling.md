# Styling

- Tailwind CSS v4 with PostCSS config.
- Design primitives under `src/styles/` and `src/styles/font.ts`.
- UI built on Radix UI + shadcn-style primitives in `src/components/ui` and `src/components/shared`.

## Tips
- Prefer `cn` helper to compose classes (if present).
- Co-locate component styles.
- Keep animations in `src/lib/animation/`.
